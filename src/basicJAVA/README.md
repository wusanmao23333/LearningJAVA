### 实验一 判断闰年
#### 实验要求  

编写Java程序，输出从公元1990年到2007年所有
闰年的年号，每输出两个年号换一行。判断年号是
否为闰年的条件是：  
（1）若年号能被4整除，而不能被100整除，则是闰年；  
（2）若年号能被400整除也是闰年。

#### 实验要求

等级与百分制对照  
*优：[90,100]  
*良：[89,80]  
*中：[79,70]
